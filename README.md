# SpeedwaBot
Simple bot de Discord sobre Jojo's hecho con NodeJS.

## Funcionamiento
`!jojo help` :

> SpeedwaBot por [@TheDavidDelta]()  
> Escribe **!jojo** para recibir la imagen de un jojo aleatorio (seasons 1-8).  
> También incluye a Gyro Zeppeli y algún *easter egg*.
